# Line Intersection Checker

2차원 평면에서 수평선과 수직선의 교차 여부를 판별하는 C++ 프로그램

## 📋 프로젝트 개요

### 목적
객체지향 프로그래밍의 핵심 개념을 이해하고, 클래스 설계부터 객체 생성을 통한 문제 해결까지의 전 과정을 경험합니다.

### 주요 기능
- 2차원 평면의 점(Point)과 선분(Line)을 클래스로 표현
- 사용자 입력을 통한 수평선/수직선 생성
- 입력값 검증 로직
- 두 선분의 교차 여부 판별

## 🏗 클래스 구조

### MyPoint001 클래스
2차원 평면에서 점의 좌표를 표현

**멤버 변수**
- `int iX` - X 좌표 (Private)
- `int iY` - Y 좌표 (Private)

**생성자**
- `MyPoint001()` - 기본 생성자, (0,0)으로 초기화
- `MyPoint001(int iX, int iY)` - 매개변수 생성자

**멤버 함수**
- `int getX()` - X 좌표 반환
- `int getY()` - Y 좌표 반환

### MyLine001 클래스
두 점으로 이루어진 선분을 표현

**멤버 변수**
- `MyPoint001 point1` - 첫 번째 끝점 (Private)
- `MyPoint001 point2` - 두 번째 끝점 (Private)

**생성자**
- `MyLine001()` - 기본 생성자
- `MyLine001(MyPoint001, MyPoint001)` - 매개변수 생성자

**멤버 함수**
- `bool isVertical()` - 수직선 여부 판단
- `bool isHorizontal()` - 수평선 여부 판단
- `string getCoordinatesP1()` - 첫 번째 점 좌표 문자열 반환
- `string getCoordinatesP2()` - 두 번째 점 좌표 문자열 반환
- `int getX1()`, `int getY1()`, `int getX2()`, `int getY2()` - 각 좌표값 정수 반환

## 🛠 기술적 특징

### 입력 검증
- 같은 점 검증: 두 점이 동일한 경우 재입력 요구
- 수평/수직 검증: 조건을 만족하지 않으면 재입력 요구

### 교차 판별 알고리즘
```
수평선: y좌표 고정, x좌표 범위 [xMin, xMax]
수직선: x좌표 고정, y좌표 범위 [yMin, yMax]

교차 조건:
- 수직선의 x ∈ [수평선의 xMin, xMax]
- 수평선의 y ∈ [수직선의 yMin, yMax]
```

### 설계 특징
- 캡슐화: 멤버 변수를 private으로 보호
- 정보 은닉: getter 메서드를 통한 접근
- 생성자 오버로딩: 다양한 초기화 방식 지원
- 효율성 개선: 문자열 파싱 대신 정수 직접 반환 메서드 추가
